Local Games — select a game to play on this PC
Files are loaded from the same folder as this launcher.
Choose a game
Select and load a local HTML game in the panel to the right.
Go (local) — `go.html`
Tic Tac Toe — `tictactoe.html`
Tip: If a game doesn't load due to local file restrictions, open the file directly in your browser or serve the folder with a small HTTP server (Python).
Quick server (optional)
In PowerShell, run the folder as a local server and open the launcher:
cd 'c:\\Users\\HChou\\Desktop\\Common\\MileStones\\Test'
python -m http.server 8000; Start-Process http://localhost:8000/games.html
